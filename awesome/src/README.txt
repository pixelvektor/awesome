Awesome App - ReadMe

Um das Programm zu starten muss die Datei Amesome.java im Hauptverzeichnis vorerst kompiliert werden.
Beispielsweise mit >javac -encoding latin1 Awesome.java< auf der Kommandozeile. Der Encoding-Parameter wird aufgrund der
verwendeten Umlaute in Labeln benoetigt. (Natuerlich ohne ><)

Anschliessend das Programm ueber die Klasse Awesome starten.
Beispielsweise mit >java Awesome< ebenfalls auf der Kommandozeile. (Auch ebenfalls ohne ><)
