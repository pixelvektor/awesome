Awesome App - ReadMe

Um das Programm zu starten muss die Datei Amesome.java im Hauptverzeichnis kompiliert werden.
Beispielsweise mit >javac Awesome.java< auf der Kommandozeile. (Natürlich ohne ><)

Anschliessend das Programm über die Awesome Klasse starten.
Beispielsweise mit >java Awesome< ebenfalss auf der Kommandozeile. (Auch ebenfalls ohne ><)

Der Code kann auch in Eclipse importiert werden und von dort aus bequem ausgeführt werden. Wie das geht, weiß Tante Google.
